## 1. 运动设计的目标

摄影机运动必须至少完成一项变化：重新分配信息、改变人物距离、重写空间关系、引导视线、承接动作能量、制造或释放压力。不要把“画面感”简化为不断移动。

强画面感通常来自四层共同变化：

1. 摄影机位置发生位移；
2. 取景方向或焦点随主体修正；
3. 前中后景产生视差、遮挡或尺度变化；
4. 运动在明确构图终点停住、反转或切出。

## 2. 运动强度等级

- `M0 锚定`：固定机位；靠人物、环境、焦点或景深变化产生内部运动。
- `M1 微动`：短距离微推/微拉、轻摇、轻俯仰、焦点转移；用于表演和信息靠近。
- `M2 叙事运动`：跟随、侧移、后退、前进、升降、遮挡穿越；明显改变空间理解。默认动态等级。
- `M3 强调运动`：弧移、越肩超越、急推后锁定、快速升降、甩摇接物、运动速度显著变化；每个短段优先只设一个核心节点。
- `M4 奇观运动`：大范围一镜多段、完整环绕、穿越式飞行、复杂变速与多次焦点交接；仅在世界观、动作高潮或明确风格需要时使用。

对 15 秒动态场景，通常使用 `M1/M2 → M2 → M3 → M0/M1`。不要五镜全部同强度，也不要每镜都 M3。

## 3. 单镜运动组件

### 平台位移（改变摄影机位置）

- 前移/推轨：缩短与主体距离，压缩犹豫或强调信息；
- 后移/拉轨：扩大空间、暴露危险或表现疏离；
- 横移/侧跟：制造前景视差、展示路径与环境；
- 前后跟随：维持主体尺度，同时让空间持续展开；
- 弧移/半环绕：重写人物与背景关系，适合决定或权力变化；
- 升/降：改变地理信息、地位或坠落感；
- 越肩超越：摄影机从人物后方追上并超过，完成主客观或目标交接；
- 穿越/遮挡移位：借门框、岩壁、人群或前景物完成揭示与转场。

### 取景修正（不改变摄影机位置）

- 摇摄 Pan：跟随水平视线或动作；
- 俯仰 Tilt：连接人物、地面、悬崖、天空或垂直动作；
- 滚转 Roll：制造失衡；必须极少且有明确心理/物理理由；
- 焦点转移 Rack Focus：改变信息层级，不要误标为位移；
- 变焦 Zoom：改变视角覆盖但不改变透视；除风格化或报道感外慎用。

### 质感与承载

- 稳定轨道/稳定器：路径清晰、空间流畅；
- 受控手持：呼吸、危险和即时反应；
- 肩扛/贴身跟随：身体性更强；
- 摇臂/升降：垂直揭示或规模变化；
- 主观/半主观：限制信息并把观众绑定人物感知。

## 4. 单镜组合运镜库

一个可读组合通常只包含：`一个平台位移 + 一个摇/俯仰/焦点修正 + 一次速度变化`。超过这一上限，升级为关键镜头并写分段路径。

### C01 后退跟随 + 轻摇锁定

摄影机面向人物后退，轻摇保持眼睛或胸口在构图锚点。适合人物逼近、压迫或进入未知空间。终点必须是门、边缘、证据或另一人物进入构图。

### C02 侧向跟移 + 前景视差揭示

与主体同向侧移，让墙、树、柱、岩石从前景滑过，逐步露出目标。适合走向危险、潜行、关系并行和环境展示。

### C03 后方跟随 + 越肩超越

先贴后跟随，再从一侧超过肩部，把主体背后的未知目标变为正面信息。适合人物抵达、真相显现或主观转客观。

### C04 缓推 + 弧移四分之一圈

一边靠近一边改变人物与背景关系，速度克制，弧度通常不超过 45-90 度。适合决心形成、权力翻转或关系重构。

### C05 后拉 + 轻升揭示

从中近景后退并略升，把孤立人物放回巨大环境。适合暴露危险、展示代价、失败或结尾余波。

### C06 下降 + 前推逼近

从较高机位下降，同时略向人物靠近。适合从地理信息过渡到人物选择，或从全局进入危机核心。

### C07 上升 + 后拉扩张

人物尺度迅速变小，环境接管画面。适合事件后果、规模揭示、结尾或不可逆选择。

### C08 低位侧跟 + 俯仰上扬

低机位跟随脚步或轮胎，随后仰起连接身体/目标。适合行走、奔跑、登阶、起跳准备和人物登场。

### C09 贴身跟随 + 急停锁定

摄影机与主体共同加速，在动作或信息命中点同步急停。适合追逐、突然决定、危险边缘和动作中断。

### C10 甩摇 + 新主体锁定

由声音、动作或视线触发快速摇摄，在新主体上明确停住。用于突发事件、反击或空间内快速因果；不要连续滥用。

### C11 推进 + 焦点前后交接

摄影机向前移动，同时焦点从前景道具转向人物或反向。适合证据与反应绑定。两层信息必须在同一空间关系中成立。

### C12 遮挡穿越 + 方向匹配

摄影机跟随主体经过完整遮挡物，在遮挡最满处切换机位或时间。适合无痕转场、空间压缩或场景切换。

### C13 反向拉近（Dolly Zoom）

推轨与反向变焦抵消主体大小、改变背景透视。仅用于强烈认知冲击或空间眩晕；一个项目中应非常稀缺，AI 轨道优先后期完成。

### C14 环绕 + 主体反向转身

摄影机与人物相反方向运动，以更快速度改变背景关系。用于决断、对峙或力量爆发。必须先锁轴线和终点背景，避免无目的整圈环绕。

## 5. 跨镜运动句法

不要逐镜随机挑运镜。先选整段句法，再分配单镜。

### P01 逼近句

`静态地理锚点 → 侧跟产生视差 → 后退跟随缩短距离 → 微推命中表演 → 急停/固定结果`

适合人物接近悬崖、房门、敌人、证据或不可逆决定。

### P02 揭示句

`受遮挡横移 → 顺视线摇摄 → 推进并焦点交接 → 升拉揭示全貌 → 静态消化`

适合秘密空间、危险规模、人物登场和真相揭示。

### P03 爆发句

`固定蓄力 → 突然前冲跟随 → 平行侧跟 → 甩摇接结果 → 动切静钉住后果`

适合奔跑、攻击、冲门、起跳和突发动作。

### P04 权力翻转句

`稳定双人关系镜 → 缓推弱势方 → 四分之一弧移重组背景 → 反向后拉强势方 → 新关系镜锁定`

适合对话对峙和立场逆转。

### P05 疏离句

`近距离跟随 → 运动逐渐落后 → 前景障碍进入 → 后拉扩大空隙 → 人物离开后静留空间`

适合离别、失败、孤独和关系破裂。

### P06 失衡句

`稳定建立 → 受控手持靠近 → 轻微下降/偏轴 → 快速方向修正 → 恢复稳定或彻底断裂`

适合危机、眩晕和主观失控。偏轴与手持幅度必须受控。

### P07 坠落/纵向句

`低位跟脚 → 仰起连接身体 → 后拉显露边缘 → 侧向加速跟随起跳 → 切至高位俯跟或留空画框`

适合登高、跳跃、坠落。若段尾停在离地瞬间，让最后镜运动在离地时停止或让人物冲出画框，不提前展示落点。

## 6. 类型化运动组合

### 行走接近危险

优先 `P01 + C02 + C03/C09`。让危险在背景或前景中逐步增长，不要五镜都固定侧面拍走路。

### 对话压迫

优先关系镜固定、弱势方后退跟随、强势方缓推、权力变化时 C04。避免每句台词都推近。

### 追逐/动作

先用地理锚点，再用贴身跟随与平行侧跟；冲击点动切静，结果后用后拉或升拉恢复地理。不要复杂动作与复杂环绕同时进行。

### 情绪重场

以 M1 为主：短横移、微推、焦点交接、运动同步停止。只在决定形成或关系翻转时出现一次 M2/M3。

### 悬疑揭示

使用遮挡横移、顺视线摇摄、焦点前后交接和缓慢升降。运动终点必须交付新信息。

### 竖屏短剧

优先纵深前后运动、低位仰起、人物前后层调度和竖向升降；减少大范围横移，避免主体长期离开中央安全区。

## 7. 动态构图与视差

每个 M2 以上镜头至少规划一项动态层次：

- 前景擦过：门框、树、岩石、人物肩部；
- 中景交接：主体被另一人物或物体短暂遮挡；
- 远景变化：目标、出口、悬崖、敌人逐步显露；
- 尺度变化：主体由小变大或由大变小；
- 构图换位：主体从边缘移至中心，或由中心被挤向边缘；
- 高度变化：地面、人物与天空/深渊形成垂直连接。

没有层次变化的移动容易像机械平移。先布置前中后景，再决定轨迹。

## 8. 连续性与剪辑

- 记录人物运动向量和摄影机运动向量；同向接保持能量，反向接只用于揭示、对抗或回收。
- 动镜切动镜时，让速度、方向或构图重心至少有一项可匹配。
- 动切静用于钉住结果；静切动用于事件启动；不要每个切点都保持同一速度。
- 快速运动镜必须提供稳定起始帧或稳定结束帧，给剪辑与生成留余量。
- 组合运镜的终点构图必须先于路径确定；没有终点的运动视为无效。

对话场的运动应形成段落弧线，而不是逐句分配运镜：关系镜负责空间锚定，微推用于主张逼近，短侧移或焦点交接用于信息转移，固定反应镜钉住结果。长对白可以跨多个镜头连续存在；镜头切换服务语义、反应和动作，不改变自然口语速度。

## 9. AI 风险与降级

- `M1` 通常 L1-L2；`M2` 通常 L2；`M3` 通常 L3；`M4` 默认 L4。
- AI 单镜优先一个平台位移加一个取景修正。人物动作复杂时，摄影机只保留一个主要位移。
- 弧移不稳定：降级为短侧移 + 两镜剪辑；
- 越肩超越不稳定：降级为后跟镜 + 正面承接镜；
- 升降揭示不稳定：降级为低/高两机位动作匹配；
- 甩摇不稳定：分别生成稳定起止镜，后期添加运动模糊转场；
- Dolly Zoom：分别生成推轨素材，后期完成变焦/透视效果；
- 跳跃/坠落：拆成起势、离地、空中、落点/冲击，不把复杂运镜和完整动作链塞入一镜。

降级时保留运动句法和戏剧终点，不只是把所有镜头改成固定。

## 10. 输出格式与自检

### 场景级

```text
运动强度基线：M0-M4
运动句法：P01-P07 / 自定义 PX
主导运动语言：最多两种
静止锚点：
关键强调节点：
收束方式：
```

### 单镜级

```text
运镜强度：M0-M4
运镜组合：C01-C14 / 自定义 CX
起点构图：
运动路径：
速度曲线：静止/慢/匀速/加速/减速/急停，可分阶段
运动终点：
主体运动关系：同向/反向/迎面/背离/超越/被超越/静止
运镜动机：
下镜承接：
```

### 自检

- 是否存在至少一个静止或减速节点，让强调运动有对比？
- 每个移动镜头是否有前中后景或尺度变化？
- 每个运动是否有明确终点，并在终点交付信息/情绪/动作？
- 相邻镜头的运动方向、速度和构图重心是否可剪？
- 是否把多种运镜堆进同一镜而没有阶段？
- AI 轨道的 M3/M4 是否有降级方案？
