## 4. Cut 卡

完整版字段：

```text
### S01-B01-C01
成片时码：00:00.000-00:03.200
成片时长：3.2s
长镜授权：无 / USER-REQUESTED
素材时长：5s
镜头功能：空间建立 / 关系建立
镜头职责：MASTER / SUPPORT / ACCENT / TRANSITION
节奏贡献：新信息 / 权力变化 / 动作推进 / 动作结果 / 情绪转折 / 空间重组 / 悬念牵引 / 转场推进
节奏豁免：无 / 单镜达到快速模式阈值时说明内部变化或悬念保持
画面主体：
可见动作：
人物表演：
景别/角度/画幅构图：
第一视觉点：
第二视觉点：
构图设计：重心/平衡 + 前中后景 + 线条/框架/留白
视觉动线：第一视觉点 → 第二视觉点 → 切出方向
光色/景深：
机位/焦段/景深：
运镜强度：M0-M4
运镜组合：C01-C14 / CX
起点构图：
运动路径：
速度曲线：
运动终点：
主体运动关系：
运镜动机：
人物调度/屏幕方向：
台词/声音：DIA / OS / VO / AMB / SFX / MUS / SIL
剪辑点：为何此刻切
承接：动作/视线/空间/声音/形状/光暗
连续性：位置、朝向、持物手、道具、服化、伤痕、光线
资产绑定：
AI风险：L1-L4 + 原因
降级方案：
```

简版仍必须保留：镜号、时码/时长、功能与职责、节奏贡献、画面动作、构图设计、视觉动线、摄影机、结构化运镜、声音、承接、连续性。不存在的字段写“无”，不要省略导致误判。结构化运镜至少写 `强度｜组合｜路径｜速度｜终点｜动机`。`COMMERCIAL_FAST` 中单镜达到 6 秒时必须填写节奏豁免。

## 6. 宫格图交付

```text
【GRID-P01｜6 格｜S01-B01】
格 1｜S01-B01-C01｜00:00.000｜关键姿态
画面：
人物/摄影机运动箭头：
屏幕方向/视线：
台词/声音摘要：
连续性与资产：
```

宫格提示词应描述单个关键帧，不把镜头全过程塞进一格。

## 7. 审计格式

```text
[ERROR|WARN|INTENTIONAL] S01-B02-C03 → C04
层级：故事 / 空间 / 动作 / 状态 / 声音 / AI
证据：
影响：
修复：
```

严重度：

- `ERROR`：导致剧情错误、无法剪接、镜号/时码冲突或明显生成失败；
- `WARN`：可执行但风险较高或依赖未声明条件；
- `INTENTIONAL`：有意违例，已写目的、预备与恢复。
