## 1. 分镜的职责

专业分镜不是“把每句剧本配一张图”，而是同时解决四件事：

- 叙事：观众何时得到哪条信息；
- 表演：人物目标、阻力和行为如何可见；
- 空间：人物、道具、摄影机和光源的关系；
- 剪辑：每个镜头为何开始、为何结束、如何接下一个镜头。

对真人拍摄，分镜要给现场留下 coverage 与剪辑余量；对动画/AI，分镜还承担资产、姿态、状态和生成单元规划。

## 2. 戏剧节拍与镜头功能

### 节拍判据

出现以下任一变化时考虑新 beat：

- 人物目标改变；
- 权力关系改变；
- 新信息被揭示或被误读；
- 行动从准备进入执行或结果；
- 情绪从控制进入裂缝、爆发或余波；
- 场景空间或观察视点发生有意义变化。

### 镜头功能词表

通用：空间建立、关系建立、人物登场、主观视点、说话人、听话人反应、信息揭示、动作、动作结果、表演细节、插入、伏笔、余波、转场。

商业增强：钩子、蓄力、释放、卡点、误导、揭底、追更悬念。

每镜最多一个主功能和一个副功能。把“画面丰富”“电影感”“节奏更好”视为无效功能描述。

## 3. 场面调度优先

按以下顺序设计：

1. 人物在追求什么；
2. 人物靠近、远离、遮挡、占据或让出什么；
3. 谁看谁，谁拒绝看谁；
4. 人物与门、桌、武器、证据、光源的关系；
5. 摄影机从哪个观察位置最清楚地表达上述关系。

空间圣经至少记录：平面结构、出入口、角色初始位、关键道具、主轴线、主要屏幕方向、光源方向、允许机位区、危险越轴区。

## 4. Coverage 设计

### 对话场

最低 coverage 不是机械正反打，而是：

1. 关系镜：交代距离、地位、遮挡和共同空间；
2. 说话人：承载需要读口型或观察策略的台词；
3. 听话人：承载台词真正造成的效果；
4. 双人/移动关系镜：权力改变时重申新关系；
5. 插入：只在道具、手势或空间信息改变理解时使用；
6. 余波：允许关键台词之后的反应完成。

不要要求“台词镜与反应镜必须一一成对”。选择更有戏剧价值的一侧；必要时让台词留在画外，把画面给反应。

### 长对白 coverage

先按自然语言、停顿和表演核算整段对白时长，再按语义主张、听话人反应、关键手势、视线变化、道具信息和反问/笑点落点拆视觉覆盖。台词可通过 OS、L-cut 或 J-cut 跨镜保持自然连续；拆镜不得造成抢词、重复完整动作或无意义正反打。

竖屏商业短剧推荐单镜 2-6 秒；7 秒允许但必须复核；8 秒及以上必须拆 coverage。长对白、焦点变化、轻微表演、轻微推拉或构图内部的小幅变化都不能自动豁免。只有用户明确指定长镜头，并在镜头卡写入 `长镜授权：USER-REQUESTED`，才允许保留 8 秒及以上单镜。其他类型按场景判断，不机械套用该硬阈值。

### 动作场

用动作因果链设计，而非只追求快切：

`地理建立 → 意图/目标 → 起势 → 路径 → 接触 → 结果 → 反应 → 新地理`

每次改变位置、武器、攻守权或目标后，重新给观众空间锚点。打击镜要能看清施力源、方向和结果中的至少两项；无法同镜稳定表达时拆镜。

### 长镜头

长镜不是“持续运镜”。写清：起始构图、主体调度、每次构图重组、焦点交接、信息揭示点、节奏停顿、结束构图和失败时的 coverage 备份。

## 5. 镜头弧线

镜头选择要形成段落级变化：

- 景别弧线：全 → 中 → 近，或近 → 全，以信息与情绪为动机；
- 角度弧线：关系稳定时保持，权力改变时才明显变化；
- 运动弧线：稳定 → 跟随 → 推进 → 静止余波，或反向设计；
- 焦点弧线：环境 → 人 → 物，或通过焦点转移改变信息优先级；
- 声音弧线：环境底 → 主体声 → 抽离/静默 → 声桥。

避免逐镜“换一种运镜”。连续固定镜头可以成立，前提是人物、构图层次、声音或剪辑点持续产生变化。

固定镜头不是安全默认值。连续两个固定镜头必须承担不同信息；连续三个固定镜头必须说明整体节奏目的，否则至少替换一个为有叙事动机的微推、短侧移、跟随、俯仰或焦点交接。增加运动时必须回答“运动改变了什么信息、距离或关系”，不能用机械微推伪装叙事感。

## 6. 类型化策略

### 克制叙事

- 允许画外空间、停顿和不完整反应；
- 用微调度和声音承载变化；
- 特写必须稀缺，否则失去强度。

### 商业短剧/竖屏

- 竖向构图优先面部、手部和前后景层次；
- 尽早建立人物关系和核心矛盾；
- 钩子来自未完成因果或新信息，不靠无逻辑惊讶表情；
- 快节奏仍要保留动作结果和关键反应。

启用 `COMMERCIAL_FAST` 时，再执行第 8 节的镜头预算、晚进早出和牵引切点规则。

### 悬疑/惊悚

- 设计观众知道、人物知道和镜头允许看见的差值；
- 通过遮挡、画外声、受限视点和延迟揭示控制信息；
- 不用随机晃动代替紧张。

### 情绪重场

- 将抽象情绪转成眼神目标、呼吸、下颌/手部张力、重心和停顿；
- 设计“控制 → 裂缝 → 决定性动作 → 余波”；
- 情绪镜不一定是脸，相关物件、距离变化和未完成动作也可承载。

## 7. 专业判断规则

- 焦段是空间关系和透视选择，不是情绪标签。若场地、传感器未知，用广角/标准/中长焦加意图描述，避免伪精确。
- 光圈取决于画幅、焦距、距离、光照和所需景深。未知条件下写“浅/中/深景深”和焦点对象。
- 机位高度与角度先服务信息可读性，再服务权力暗示。
- 手持不等于真实，稳定器不等于高级；运动质感必须与人物状态、场景材质和剪辑节奏一致。
- 特写、升格、荷兰角、环绕、快速推拉都是高强调资源，必须节制使用。

## 8. 商业快节奏切分

快节奏不是把每句话切碎，而是减少观众等待“已知结果”的时间。按以下顺序压缩：

1. 删除可推断的准备、走位、寒暄、完整收势和重复反应；
2. 将空间建立合并进人物进入、动作启动或线索揭示；
3. 让声音跨切，画面提前去往结果、反应或下一目标；
4. 把结果完全落定前的动作、视线、问题或声音交给下一镜；
5. 最后才缩短关键表演、动作接触或观众理解所需时长。

普通 beat 默认使用 1 个 MASTER 和最多 1 个 SUPPORT；需要动作接触、信息标点或空间重定位时可加 1 个 ACCENT。反应镜必须至少完成一项变化：形成决定、暴露立场、改变权力、提供新信息或启动下一动作。仅表示“听见了、惊讶了、看了一眼”的反应留在同镜或删除。

每镜写 `节奏贡献`：新信息、权力变化、动作推进、动作结果、情绪转折、空间重组、悬念牵引或转场推进。连续两镜贡献相同且第二镜没有增加强度、方向或后果时，合并或删除。

快速模式仍保留必要的动作结果与关键反应，但不要求每个微事件完整经历“建立—意图—执行—结果—反应—余波”。根据观众已掌握的信息跳过可推断环节；在不可逆变化上命中，在下一结果完全关闭前离开。
