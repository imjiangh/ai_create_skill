## 1. 构图的任务

构图不是把人物摆得“好看”，而是控制观众注意力、关系判断和情绪距离。每镜先回答：

1. 第一眼必须看见什么；
2. 第二眼需要发现什么；
3. 视线通过什么线条、明暗、焦点、动作或运动完成转移；
4. 哪些区域必须保持安静；
5. 镜头结束时画面关系如何变化。

美感来自层级清楚、力量关系明确、空间有深度、形状与光色协调。对称、三分法、黄金分割只是工具，不是质量证明。

## 2. 场景视觉圣经

每个 Scene 或长段先定义：

```text
构图母题：对称秩序 / 对角冲突 / 框中框困束 / 大留白孤立 / 纵深追逐 / 自定义
主导几何：水平 / 垂直 / 对角 / 三角 / 圆弧 / 放射
人物关系形状：并列 / 对峙 / 包围 / 前后层级 / 高低层级
景深策略：深景深空间叙事 / 浅景深主观隔离 / 由深转浅 / 由浅转深
负空间方向：人物视线、运动或威胁的预留方向
主要光色分区：亮区/暗区、冷区/暖区分别属于谁或什么
视觉母题：门框、镜面、海平线、栏杆、圆形灯、红色物件等
构图破格节点：何时有意打破此前秩序
```

同场景不必每镜相同，但变化要形成弧线。关系或心理发生重大变化时，才明显改变母题、平衡或空间层级。

## 3. 单镜构图七要素

### 3.1 第一视觉点

用至少两种手段锁定：亮度反差、色彩反差、清晰度、运动、尺度、位置、面孔、线条汇聚。不要让三个元素竞争第一视觉点。

### 3.2 第二视觉点

承担线索、威胁、反应或环境信息。它应在第一视觉点之后被发现，通过视线、动作、焦点转移或构图线条到达。

### 3.3 重心与平衡

- 对称平衡：秩序、仪式、控制、人工感；
- 非对称平衡：用小而亮、远而动或高对比元素平衡大面积暗部/主体；
- 有意失衡：压迫、即将离开、危险逼近；必须保留明确视觉锚点。

### 3.4 前中后景

至少两层，M2 以上运动镜优先三层：前景提供框架/遮挡/视差，中景承担人物动作，远景交代目标、威胁或后果。

### 3.5 线条与形状

水平稳定，垂直威严或禁锢，对角制造能量与不安，曲线引导流动，三角形成关系层级，重复形状制造秩序或压迫。让线条引向主体，避免从头部或关节处产生背景切线。

### 3.6 留白与呼吸区

- 视线留白：给人物看向的方向；
- 运动留白：给进入、奔跑、车辆或攻击方向；
- 威胁留白：让观众预期某物进入；
- 孤立留白：让空白压过人物；
- 反向挤压：故意减少前方空间，表现受阻或走投无路。

### 3.7 视线动线

写成 `第一视觉点 → 第二视觉点 → 离开画面的方向/剪辑点`。若观众第一眼可能看错，调整光、焦点、尺寸、位置或运动，不靠文字解释。

## 4. 构图语法库

### 三分与黄金区域

适合自然、稳定而不僵硬的画面。把眼睛、地平线、关键物放在强点附近，而非机械贴线。主体与第二视觉点可落在对角强点形成眼动。

### 中心对称

适合仪式、权力、规则、凝固和异化。只有当环境本身支持对称时才使用；微小破坏对称可成为剧情信号。

### 框中框

用门、窗、拱、栏杆、镜面、车窗、前景肩部限制主体。适合窥视、困束、分隔和层层揭示。框架不可遮住关键表演区域。

### 对角线

让人物、地形、武器、视线或光带沿对角线排列，制造方向和力量。对角线起点与终点应承载因果，不只是倾斜画面。

### 三角关系

三人或人物—道具—目标形成稳定/不稳定三角。顶点通常拥有权力或注意力；关系变化时让人物离开或重组三角。

### 引导线与汇聚

道路、桌边、墙线、海岸线、光束、群众视线指向主体。避免引导线把观众带出画面或指向无意义区域。

### 大留白

主体占比小，用环境尺度、等待或孤独接管画面。留白必须有质感、方向或潜在信息，不能只是空。

### 多层纵深

通过前景遮挡、中景人物、远景目标和不同焦平面建立深度。适合悬疑、关系复杂、环境叙事和动态摄影。

### 镜像与分割

镜面、玻璃、阴影、门缝或色块把人物分裂或并置。严格检查反射、视线和 AI 稳定性；必要时分层合成。

### 高/低地平线

低地平线强调天空、垂直和人物崛起；高地平线强调地面、路径、压迫或坠落空间。避免地平线穿过眼睛、颈部或关键动作关节。

## 5. 人物与关系构图

### 单人

- 给眼神与动作方向留空间；
- 用背景形状、亮区或道具建立第二层信息；
- 近景优先保护眼睛、嘴和手的动作，不在关节处裁切；
- 刻意压头顶、挤视线或切断身体时写明压迫目的。

### 双人

- 并列：同盟、平等、共同目标；
- 对称对峙：僵持、仪式化冲突；
- 一前一后：权力、隐瞒、情绪距离；
- 肩部框架：主观关系与空间连续；
- 中间障碍：关系阻隔；障碍必须在剧情空间中真实存在。

### 多人

先分主次群组，再构建三角、对角或前后层。不要把每张脸等大、等亮、等清晰。群像每个 beat 只保留一个主视觉中心。

### 动作人物

给运动方向留空间，让身体轮廓可读。关键动作帧优先清晰 silhouette、受力线和接触点；不要让肢体与背景线条粘连。

## 6. 景别与焦段策略

先选观众需要的空间/表演信息，再选焦段：

- 广角：强调空间关系、路径和前后景；靠近人物时会强化透视；
- 标准：关系自然，适合客观观察和中景调度；
- 中长焦：压缩空间、隔离表演、制造窥视或命运逼近；
- 微距：只用于真正改变理解的物件/身体细节。

焦段不是情绪标签。场地和传感器未知时写“广角/标准/中长焦＋空间意图”，避免毫米数伪精确。

景别弧线不必总是全→中→近。可用近→全揭示孤立、全→近逼近决定、近→近维持窒息，关键是每次变化都重新分配信息和情绪距离。

## 7. 光色、景深与材质

### 明暗层级

先保证主体与背景分离，再考虑漂亮光线。可用亮度、轮廓光、负补光、烟雾层次或局部反射分离。脸不是必须最亮，但眼睛和关键动作要可读。

### 色彩组织

- 主色决定环境基调；
- 辅色建立人物/空间归属；
- 强调色只给关键物、危险或转折；
- 情绪变化可通过色彩占比改变，而不必整镜突然换色。

### 景深

浅景深隔离主体，也可能破坏多人关系和动作可读性；深景深让空间叙事，也可能分散注意力。焦点转移必须对应信息交接。

### 材质与空气

雨、雾、尘、玻璃、金属、水面、织物和皮肤反光可建立层次，但不要堆满。选择一到两个与场景意义相关的材质母题。

## 8. 跨镜构图弧线

把构图当成剪辑序列：

- 秩序→破坏：对称逐渐偏移、亮暗失衡、人物退出原位置；
- 接近→压迫：主体比例增大、负空间缩小、背景被压缩；
- 自由→困束：开阔全景转框中框/前景遮挡；
- 隐瞒→揭示：第二视觉点逐渐变为第一视觉点；
- 孤立→连接：单人分割转双人同框或共享光区；
- 决断→后果：近景命中后切大环境，让后果接管画面。

相邻镜头可通过形状、明暗、色彩、视线位置或运动向量匹配。避免连续镜头的头部大小、主体位置和背景线条近乎相同而产生无意跳切。

## 9. 画幅适配

### 16:9 / 宽银幕

利用横向关系、运动方向、地平线和多人距离。不要把所有主体放中央；用横向负空间和前景层次组织环境。

### 9:16 / 竖屏

优先纵向层级、前后调度、门窗/建筑竖线、头手关系和上下空间。重要面部与动作保持在中央安全区；大范围横移要让主体及时回到画面。

### 1:1 / 4:5

适合中心、三角和层叠关系。环境规模需要通过上下层次、焦段或多镜建立，不能硬塞宽景。

需要多画幅交付时，先确定保护区，不能简单裁切关键人物、视线留白或动作路径。

## 10. 宫格关键帧

每格选择“信息或关系最清楚的决定性瞬间”，而非动作中最模糊的过渡帧。标注：

- 第一/第二视觉点；
- 主体与摄影机运动箭头；
- 前中后景；
- 光色与焦点；
- 屏幕方向；
- 起点帧/终点帧或关键动作相位。

同镜多个关键姿态要保持背景透视、轴线、人物比例和光源一致。

## 11. 常见失败与自检

- 所有人等大、等亮、等清晰；
- 主体永远居中，构图没有关系变化；
- 背景线穿头、地平线切颈、肢体在关节处裁断；
- 空白没有方向，视线/运动方向被画框挤死；
- 前景遮挡关键表演或抢成第一视觉点；
- 浅景深导致关键第二信息不可读；
- 强调色在无关物体上抢注意力；
- 每镜换一种构图，没有场景母题；
- 构图漂亮但无法解释戏剧功能；
- 动态镜只有摄影机位移，没有层次、视差和终点构图。

交付前逐镜回答：第一眼看哪里？第二眼看哪里？为什么？画面力量朝哪里？下一镜如何接走这股力量？
