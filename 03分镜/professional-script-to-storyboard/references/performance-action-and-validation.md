## 1. 双视角分析

### 演员视角

- 人物此刻要得到什么；
- 表面行为与真实意图是否一致；
- 哪个身体部位最先泄露；
- 哪个动作正在被压制；
- 呼吸、视线、重心如何变化。

### 观众视角

- 此刻必须知道什么；
- 应该误读或暂时不知道什么；
- 第一眼、第二眼分别看什么；
- 哪个动作/表情/物件必须被看清；
- 情绪命中发生在哪个镜头和哪个瞬间。

两者不对齐时，先调整构图、表演、声音和景别；只有确有信息缺口时补镜。

## 2. 情绪与视听锚点

每个关键 beat 建立：

```text
戏剧动作：
情绪：起点 → 对抗/过渡 → 落点
视觉锚点：脸 / 手 / 道具 / 距离 / 空间 / 光色
表演锚点：一个主要身体事实 + 一个微观事实
声音锚点：呼吸 / 动作声 / 环境底 / 静默 / 音乐变化
决定性瞬间：
```

锚点是方向，不是固定镜头套餐。根据场地、角色、画幅和制作轨道重新设计。

## 3. 可见表演三层

### 宏观层

重心、距离、身体朝向、靠近/远离、站/坐/跪、占据或让出空间。

### 中观层

手部张力、肩颈、胸腔、头部角度、视线目标、拿放道具、动作速度和停顿。

### 微观层

眨眼、眼眶湿度、嘴角控制、下颌张力、吞咽、鼻翼、呼吸、皮肤与泪痕状态。

每镜优先一个主层和一个辅助层。不要在单镜同时指挥全身、双手、十几块面部肌肉和复杂运镜。

心理词必须落成可见/可听事实。例如：

- 隐忍：下颌锁住，嘴唇压紧，眼眶水光增加，吞咽后仍不说话；
- 焦虑：视线在两个目标间跳转，拇指反复摩擦食指，浅短胸式呼吸；
- 假笑：嘴角上扬但眼周不参与，笑容结束得过快；
- 释怀：肩膀随完整呼气下降，手指松开，视线不再回避。

仅在特写、医学/表演教学或用户要求时使用肌肉名称；普通分镜优先写视觉结果，避免解剖术语淹没表演。

## 4. 情绪转换与呼吸

用三阶段描述：

`起幅状态 → 对抗/泄露 → 落幅状态`

过渡不应直接跳变。写出哪个控制动作失败、哪个新信号接管。例如“抿嘴维持笑容 → 嘴角轻颤且呼吸卡住 → 笑容消失、肩膀下沉”。

呼吸选择：

- 平稳腹式：平静、控制、自信；
- 浅短胸式：紧张、焦虑；
- 屏息：突然发现、恐惧、等待；
- 卡喉/吞咽：隐忍、难以开口；
- 长呼气：释怀、疲惫、决定完成；
- 口鼻交替/过度换气：高唤醒危机。

近景看喉结、嘴唇和鼻翼，中景看胸腔和肩膀，全景看姿态和节奏。声音要与可见呼吸相容。

## 5. 主辅标点镜分工

- `MASTER`：承担 beat 的核心动作、信息或情绪，删掉则 beat 不成立；
- `SUPPORT`：补空间、反应、道具、表演细节或连续性；
- `ACCENT`：短促命中声音、物件状态或微反应，改变节奏；
- `TRANSITION`：承担时间、空间、视点或声音跨越。

每个 beat 先放 MASTER，再只添加必要 SUPPORT。ACCENT 必须有明确标点对象，不能用随机特写制造“电影感”。快切素材也必须独立镜号、完整状态和剪辑点。

`COMMERCIAL_FAST` 下，普通 beat 默认只允许 1 个 MASTER 和最多 1 个 SUPPORT；第三镜只能是具有新信息、动作结果或情绪命中的 ACCENT。反应若不改变选择、权力、信息或下一动作，不独立成镜。情绪表演可以在声音跨切中继续，不必等脸部反应完整结束后再切。

## 6. 动作场六层链条

### 1. 动机与目标

谁要到达、阻止、夺取、逃离或保护什么。动作必须改变局势。

### 2. 力学与阶段

记录准备、发力路径、接触/错过、结果和恢复。说明力量来源、方向、身体重心与受力对象。

### 3. 对手/环境反应

受击、闪避、阻挡、失衡和环境反馈必须符合方向与强度。动作结果不能只靠攻击者表演。

### 4. 道具/武器状态

记录持物手、朝向、开合、弹药、破损、污渍、掉落位置和交接过程。

### 5. 摄影机与空间

先建立地理，再跟随意图；冲击点选择看清路径或结果，呼吸点恢复关系镜。每次位置或攻守权改变后重建空间。

### 6. 声音与节奏

脚步、衣物、呼吸、器械、接触、环境反射和音乐节拍共同建立重量。不要用夸张冲击声掩盖错误物理。

动作循环可用：`准备 → 执行 → 接触/闪避 → 反应 → 恢复/新局势`。AI 轨道把复杂链条拆成独立生成单元。

## 7. 类型化映射使用方法

可参考常见关联，但作为候选而非锁定模板：

- 仪式：对称、深景、缓升、低频；
- 孤独：小主体、大留白、环境声；
- 压迫：框中框、负空间被挤压、缓推或后退跟随；
- 追逐：地理锚点、跟随、视差障碍、呼吸与脚步；
- 决断：控制动作停止、物件状态改变、运动急停或声音标点；
- 关系破裂：人物分层、障碍进入、共同光区分离；
- 揭示：遮挡移除、焦点交接、第二视觉点升级为第一视觉点。

每次选择都要根据具体剧本回答“为什么适合此人、此地、此刻”。不要固定 80/15/5 权重，也不要用导演名字替代可执行描述。

## 8. 镜头价值四问

逐镜内部检查：

1. 服务哪个 beat、哪个决定性瞬间？
2. 删除后损失什么？若什么都不损失，删除或合并。
3. 是否有更清楚、更有力且更可执行的视听表达？
4. 第一视觉点、第二视觉点、运动终点和切点是否明确？

再检查红线：原文是否被改、抽象心理是否未转译、构图是否竞争、动作物理是否断裂、运镜是否无终点、AI 高风险是否无降级。
